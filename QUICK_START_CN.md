# 🚀 快速開始指南

## 1️⃣ 啟動應用 (3種方式)

### 📱 在 VS Code 中
1. 安裝 **Live Server** 擴展
2. 右鍵點擊 `index.html`
3. 選擇 "Open with Live Server"

### 🐍 使用 Python
```bash
cd /workspaces/Media-pipe
python3 -m http.server 8000
# 然後訪問: http://localhost:8000
```

### 🟢 使用 Node.js
```bash
npm install -g http-server
cd /workspaces/Media-pipe
http-server -p 8000
# 然後訪問: http://localhost:8000
```

## 2️⃣ 使用應用

| 步驟 | 操作 |
|-----|------|
| 1️⃣ | 點擊「**開啟相機**」按鈕 |
| 2️⃣ | 允許瀏覽器訪問攝影機 |
| 3️⃣ | 面向攝影機，自然表達表情 |
| 4️⃣ | 應用將自動識別並播放語音回饋 |

## 3️⃣ 設定選項

### 語音回饋 🔊
- 勾選啟用語音反應
- 取消勾選以靜音

### 顯示檢測框 🎯
- 勾選顯示臉部檢測可視化
- 取消勾選以隱藏

### 調整信心度
- 拖動滑塊 (0.3 = 敏感, 0.9 = 嚴格)
- 提高阈值可減少誤報

## ⚙️ 常見問題

**Q: 攝影機無法訪問？**
- 檢查瀏覽器隱私設定
- 確保網站在 HTTPS 上（localhost 除外）

**Q: 表情識別不準確？**
- 確保光線充足
- 保持穩定的頭部位置
- 調整信心度閾值

**Q: 語音反饋無聲？**
- 檢查系統音量
- 檢查瀏覽器音量設定
- 確認「啟用語音回饋」已勾選

## 💡 提示

1. **最佳效果**
   - 在光線充足的地方使用
   - 攝影機距離臉部 30-60cm
   - 完全展示臉部到攝影機

2. **準確識別**
   - 自然表達表情
   - 避免快速移動頭部
   - 保持相機穩定

3. **語音反饋**
   - 確保揚聲器/耳機連接
   - 調整瀏覽器音量
   - 某些語言可能有限制

## 📞 反饋和報告

如遇到問題或有建議，請：
1. 檢查瀏覽器控制台 (F12) 是否有錯誤
2. 嘗試重新加載頁面
3. 清除瀏覽器緩存
4. 使用不同瀏覽器測試
